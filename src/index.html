The Things We Do

Here's just a few of the things we do. We mainly work with small companies in the Creative and Web Industries. When not doing client work, we're working on open-source projects, which can be found on Github.

How We Work

Here, at Branded Code, we have a little bit of a theory going about how things should be done. We believe that everything should be given the time and effort required to look at both the details and the larger picture.

Instead of always producing the same old code and designs, we like to experiment to find the best solution to the problem at hand. We believe in creativity & ideas powering the work we do.
